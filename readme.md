# 🚀 Projeto API de Usuários

Este é um projeto de portfólio desenvolvido para demonstrar o consumo de APIs REST utilizando JavaScript moderno.

## 🛠️ Tecnologias
- **HTML5** e **CSS3** (Layout responsivo e variáveis)
- **JavaScript (ES6+)**
- **Fetch API** (Para buscar os dados)
- **Async/Await** (Para lidar com promessas)

## 📋 Funcionalidades
- Busca de dados reais da API JSONPlaceholder.
- Exibição de cards dinâmicos com iniciais do nome.
- Estatísticas automáticas de usuários, cidades e empresas.
- Sistema de erro caso a API falhe.
- Layout inspirado em terminais modernos.
